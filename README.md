# Animals

**Correction de l'exercice Animals**

**Énoncé**

```
Faire un mini jeu avec des animaux qui s'attaquent (Pokemon style) :
On doit pouvoir générer des animaux de certains types, leur donner un nom.
Lors de la création de l'animal, ce dernier obtiendra automatiquement un nombre d'HP et de force d'attaque (Attaque entre 0 et 30, Vie entre 400 et 800)
En fonction de l'animal, une attaque spéciale doit être possible :
    -- Chat : CalinChatPotté = 30 + (entre 0 et 20)
    -- Chien : Scoobidou = 20 + (entre 0 et 40)
    -- Furet : BoulePuante = 15 + (entre 15 et 50)
Le Chat à 1 chance sur 3 de faire son attaque spéciale

Le Chien à 1 chance sur 4 de faire son attaque spéciale

Le Furet à 1 chance sur 2 de faire son attaque spéciale

Les animaux pourront s'attaquer dans le main() ou dans le GameEngine. Attention à bien vérifier lequel a gagné.
```
